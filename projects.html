Shift Work

Shift work is a game about office life. The game was originally a school project but has become something I continue to work on. The genre is an office-based simulation where the player has to complete a task on each floor before time runs out. This game helped me improve my understanding of design patterns

Smoggle

The goal of this project is to be able to look up the air quality for various locations. To use this app, click anywhere on the map to place a marker and it will reveal the air quality for that location. The map is created using the google maps API. The air quality data is fetched from IQAir's API
