Shift Work

Shift work is a game about office life. The game was originally a school project but has become something I continue to work on. The genre is an office based simulation where the player has complete a task on each floor before time runs out. This game helped me imporve my understanding of design patterns

Smoggle

The aim of this project is to be able to look up the air quality for various locations. To use this app just click anywhere on map to place a marker which will reveal the air quality for that location. The map is implemented using google maps API. The air quality data is pulled from IQAir's API
